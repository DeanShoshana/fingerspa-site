# FingerSpa legal and support site

This public repository contains the GitHub Pages-ready privacy, terms, and support foundation for FingerSpa. `FingerSpa` is a provisional working name pending the approved name-clearance milestone.

## Publication state

The pages are intentionally marked `noindex` and GitHub Pages deployment is not enabled in PR 0. Public activation is blocked until:

- product-name clearance is approved;
- the public support email is supplied;
- country/jurisdiction is supplied;
- the exact legal seller and copyright owner are verified in Apple Team `WJ7DNHFX2W`;
- jurisdiction-specific legal review is completed or explicitly waived by the Product Director.

The legal drafts are structurally complete and need only the identified owner/contact/jurisdiction substitutions plus final counsel review. They do not replace legal advice.

## Validation

Run:

```sh
python3 Scripts/validate_site.py
```

CI performs the same validation on every pull request. Pages deployment will be added only after the publication blockers are resolved and final public branding is approved.
